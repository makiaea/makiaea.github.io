      • first draft
          • just using links to other apps and as few proprietary apps as possible
          • want to have data in rich format such as mind map that is easily manipulable
          • timing data not actually that critical but the easier it is to use the better
          • apple reminders holds core timing data, which is synced by apple icloud
          • a url scheme makes it easier to open the relevant mind map
          • fantastical2 app holds/manipulates url scheme data which apple reminders cannot currently hold, that is, it adds url data to the repeated reminder which can then be opened (albeit awkwardly) at time of reminder.
          • notifications should be persistent until processed
          • notifications are set to come from fantastical2 rather than apple reminders, so that we can open url more easily from the notification.
          • the internal apple notification system is used, so missed notifications can be processed later
          • mind map holds actual data that we update with additional data or relationship structure occasionally
          • ithoughts has a good url scheme that can open individual mind maps
          • public mind map data is synced on the “box” service so that it can be shared and updated automatically to others to access if they wish
